# Welcome to Auto Blog System

This is a sample blog post created by the auto blog system. The system will automatically generate professional blog content based on your keyword list.

## Features

- **Automatic Content Generation**: Creates professional blog posts daily
- **Keyword Rotation**: Cycles through your keyword list systematically
- **Image Generation**: Creates anti-AI detection images
- **Auto Rebuild**: Automatically rebuilds Nuxt after 10 posts
- **Cross-Platform**: Works on both Windows and Linux

## Getting Started

1. Make sure your keyword list is in `data/keywords/2025-08-21/list-keyword.txt`
2. Run `npm run auto-blog:start` to start the system
3. The system will generate 10 posts daily at 3:00 AM
4. Check status with `npm run auto-blog:status`

## Configuration

Edit `auto-blog-config.json` to customize:
- Schedule time
- Number of daily posts
- Rebuild settings
- Content quality
